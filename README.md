## ADAM - A Question Answering System. 
### Question Analysis Branch

#### Pipeline :
* Sentence Segmentation
* Tokenization
* Part Of Speech Tagging (Fine Grained)
* Lemmatization
* Named Entity Recognition
* Syntactic Dependency Parsing
